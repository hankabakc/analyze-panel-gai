## 🇺🇸 English

### 📝 Project Overview
This is the frontend application for the **ANALYZEPANEL** ecosystem. It provides a premium, data-driven experience for managing student analytics, teacher-student pairings, and AI-powered report insights.

### 🚀 Key Technologies
- **React 19.2:** Utilizing the latest concurrent features and improved hooks.
- **Tailwind CSS 4.0:** Ultra-fast, modern styling with a focus on responsiveness.
- **Recharts:** High-performance SVG charts for student progress visualization.
- **Axios:** Robust HTTP client with centralized interceptors for JWT security.

### 🔧 Installation
1. **Install Dependencies:**
   ```bash
   npm install
   ```
2. **Environment Setup:**
   Copy the example environment file and update the API URL if necessary.
   ```bash
   cp .env.example .env
   ```
3. **Run Development Server:**
   ```bash
   npm run dev
   ```
4. **Build for Production:**
   ```bash
   npm run build
   ```
